よくあるご質問

顔写真はサーバーにアップロードされますか？ +
いいえ。すべての顔検出・計測はお使いのデバイス上で完結します。顔画像がお使いのスマートフォンから外に出ることは一切ありません。サーバーに送信されるのは計算後の数値結果のみです。
計測頻度や推奨タイミングはありますか？ +
1日1回、同じ条件下（例：朝の洗顔後、自然光、正面を向いた状態）での計測をおすすめします。条件を統一することで、長期トレンドデータの精度が高まります。
22項目はどのように計算されますか？ +
FaceorbitはMediaPipe Face Meshを使用して468個の顔ランドマーク点を検出し、幾何学的計算（距離・角度・比率など）で各指標値を算出します。主観的なスコアリングやフィルター効果ではなく、科学的な幾何計測です。
クラウドベンチマークデータはどこから来ていますか？ +
全ユーザーの匿名計測結果を国・性別・年齢層別に集計したデータです。すべて完全に匿名化されており、個人を特定することはできません。
サブスクリプションをキャンセルするには？ +
iPhoneの設定 → Apple ID → サブスクリプション → Faceorbitを選択 → キャンセルしてください。現在の請求期間が終了するまでは引き続きご利用いただけます。
データをすべて削除するには？ +
アプリ設定の「すべての記録を削除」を選択するか、faceorbittw@gmail.comまでご連絡ください。7営業日以内に対応いたします。
対応言語は？ +
繁体字中国語・英語・日本語・韓国語に対応しています。
