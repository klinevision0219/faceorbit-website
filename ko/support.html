자주 묻는 질문

얼굴 사진이 서버에 업로드되나요? +
아닙니다. 모든 얼굴 감지·측정은 귀하의 기기에서 완전히 처리됩니다. 얼굴 이미지가 스마트폰 밖으로 나가는 일은 없습니다. 서버로 전송되는 것은 계산된 수치 결과뿐입니다.
측정 빈도나 권장 시간이 있나요? +
하루 1회, 동일한 조건에서（예: 아침 세안 후, 자연광, 정면을 바라본 상태）측정하시길 권장합니다. 일관된 조건이 장기 트렌드 데이터의 정확도를 높입니다.
22가지 지표는 어떻게 계산되나요? +
Faceorbit은 MediaPipe Face Mesh를 사용하여 468개의 얼굴 랜드마크 포인트를 감지하고, 기하학적 계산（거리·각도·비율 등）으로 각 지표값을 산출합니다. 주관적 점수나 필터 효과가 아닌 과학적 기하 측정입니다.
군중 벤치마크 데이터는 어디서 오나요? +
모든 사용자의 익명 측정 결과를 국가·성별·연령대별로 집계한 데이터입니다. 모두 완전히 익명화되어 있어 개인을 특정할 수 없습니다.
구독을 취소하려면 어떻게 하나요? +
iPhone 설정 → Apple ID → 구독 → Faceorbit 선택 → 구독 취소. 현재 결제 기간이 끝날 때까지 계속 이용하실 수 있습니다. 모든 구독은 Apple을 통해 관리됩니다.
모든 데이터를 삭제하려면 어떻게 하나요? +
앱 설정에서 '모든 기록 삭제'를 선택하거나 faceorbittw@gmail.com으로 연락 주세요. 7 영업일 이내에 처리해 드립니다.
지원 언어는? +
번체 중국어·영어·일본어·한국어를 지원합니다.
